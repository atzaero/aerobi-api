# Bandeiras dos estados (assets do seed)

PNGs das bandeiras das 27 unidades federativas (`<UF>.png`), usados pelo runner
de seed `states` como imagem de cada grupo de estado.

- **Fonte**: [bgeneto/bandeiras-br](https://github.com/bgeneto/bandeiras-br) (pasta `imagens/`).
- **Domínio público**: bandeiras de estados são símbolos oficiais e de domínio
  público no Brasil.

Para atualizar, rebaixe de `imagens/<UF>.png` mantendo o nome em maiúsculas
casando o enum `Uf` do Prisma. O upload aceita apenas PNG/JPEG/WebP até 5 MB.
